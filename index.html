Handy Dandy Countdown Timer.

This program was created using HTML, CSS, & JavaScript. Select a sound to play when the timer ends, or uncheck the box for no sound. Once you have selected a sound, click a time to start the countdown. The outdated appearance is intentional - I found this program (from 2003) burried in a shared drive and decided it would be a good way to test my abilities on recreating something from visuals and interaction only. Check out the README on github for images of the original project.
